# ひとこえ 提出ファイル説明

このフォルダは、運動提案Webアプリ「ひとこえ」の提出用データ一式です。

「ひとこえ」は、アプリを開くだけで今日おすすめの運動を表示し、準備なしですぐ運動を始められることを目的としたWebアプリです。ログイン機能やGoogle連携は使用していません。

## デモURL

Web上で確認する場合は、以下のURLを開いてください。

```text
https://ykt72.github.io/hitokoe/
```

## 起動方法

### Webで確認する場合

以下のURLをブラウザで開きます。

```text
https://ykt72.github.io/hitokoe/
```

### ローカルで開発版を起動する場合

プロジェクトフォルダ内で以下を実行します。

```powershell
npm install
npm run dev
```

表示されたURLをブラウザで開きます。通常は以下のようなURLです。

```text
http://localhost:5173
```

### 履歴をサーバー側にも保存して確認する場合

```powershell
npm run app
```

通常は以下のURLで開けます。

```text
http://localhost:4173
```

## 動作条件

| 項目 | 内容 |
|---|---|
| 必要な機器 | インターネットに接続できるパソコンまたはスマートフォン |
| 対応OS | Windows、macOS、iOS、Androidなど |
| 推奨ブラウザ | Google Chrome、Microsoft Edge、Safariなどの最新版 |
| 開発時に必要なもの | Node.js、npm、Git |
| ログイン | 不要 |
| Google連携 | 使用しない |

## アプリの主な機能

- ホーム画面で今日おすすめの運動を表示
- 低・中・高の強度別に運動を提案
- 運動時間をスライダーで選択
- タイマー画面で開始、一時停止、再開、終了を操作
- 完了・未達成の記録を保存
- 記録画面で運動履歴や18種類チャレンジの進行状況を確認
- 設定画面でダークモード、背景色、背景パターン、背景パターン色を変更
- PC表示とスマートフォン表示の両方に対応

## 補足

`node_modules/` はnpmパッケージが入るフォルダです。環境によっては容量が大きいため、提出方法によっては含めない場合があります。その場合でも、`npm install` を実行すれば再作成できます。

`work/` や `outputs/` は作業中に生成した補助ファイルを置くためのフォルダです。アプリ本体の実行に必須ではありません。
